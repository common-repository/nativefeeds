Customise WordPress site with product offers feeds.

== Description ==

NativeFeeds WordPress Plugin is a definite answer to your desire of having your own, fully functional coupon code site.

* Shortcodes can be used anyway you would like.
* Feeds are been managed automatically.
* The feeds provided are highly responsive and attractive, that will gain the attraction of the customers

== Installation ==

1. Upload 'nativefeeds' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Click on the new menu item "NativeFeeds" to set up the key!
4. provide your key and save it!
5. Read the documentation to enjoy the feeds:
